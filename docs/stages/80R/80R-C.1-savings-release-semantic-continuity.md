## Reserve-class language table

| copyKey / class | Modal title (example) | previewLine | tradeoffLine |
|-----------------|----------------------|-------------|--------------|
| **continuity** / T0 | Use part of your buffer | This buffer can support the month if you choose. | — |
| **flexible** / T1 | Use part of this reserve | This can give the month a little more room. | — |
| **tradeoff** / T2 | Use part of this saved amount | This can support the month if you choose. | May trade against what this was set aside for. |
| **future_capital** / T3 | Use part of this investment reserve | This can support the month if you choose. | Would trade against longer-term growth. |

Plan CTA uses neutral `planCtaLabel` per tier when default source is tier-classified at open; generic fallback: **Use part of a reserve**.

---

## Truth line law

Every applicable preview includes:

1. **truthLine:** “This does not change Monthly Left. It records that part of your reserve supported the month.”
2. **reserveReduceLine** (when amount > 0): “Your reserve balance will reduce by £X.”
3. **planTruthLine** (when overbudget): “The plan still reflects the month as it stands.”

**Removed:** “Pressure covered”, “Remaining pressure”, implied affordability repair.
