# Stage 80M-C.1 — Month Context Skip Behaviour Audit

**Type:** Read-only design audit  
**Verdict:** NEEDS ATTENTION  
**Date:** May 2026  

---

## Finding

**“Skip for now” behaved like “hide for the whole month”** when month context was empty.

| Layer | Before fix |
|-------|------------|
| Handler | `geodeDismissMonthContextForNow` set `S.monthContext.dismissedYm = currentYM()` |
| Persistence | `save()` → localStorage |
| Visibility | `mc.dismissedThisMonth && !mc.hasAny` → entire section removed |
| Toast | “Noted for this month.” — matched month-long hide, not button label |

---

## Recommendation

**80M-C.2:** Visit-scoped skip for **empty** context only — sessionStorage key `geode_month_context_skip_visit`. Keep persisted `dismissedYm` only when saved context exists and user skips (section remains visible anyway).

---

## Reflection impact

Skip does not trigger Reflection. Empty save does not trigger Reflection. No change required to Reflection pipeline.
