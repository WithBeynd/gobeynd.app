## Changes

### Suggested gate (stricter)

Requires **all**:

- `opts.trigger === 'reality_month_context_save'`
- `opts.justSavedMonthContext === true`
- `opts.dismissedThisMonth !== true`
- existing snapshot / content / `noGo` checks

Without `justSavedMonthContext` → `available` + `suppressReason: 'missing_just_saved_flag'`.

With `dismissedThisMonth` → `available` + `suppressReason: 'dismissed_this_month'`.

### Output contract

| Field | Notes |
|-------|--------|
| `label` / `dismissLabel` | Replaces `suggestCopy` / `dismissCopy` |
| `reasonCodes` | Array of gate codes |
| `suppressReason` | Why suggest was withheld |
| `trigger` | Top-level echo of `opts.trigger` |
| `frequency` | `once_per_save_event` \| `on_demand` \| `null` |
| `audit.noHomeEntry` etc. | All `true` — shadow safety flags |
