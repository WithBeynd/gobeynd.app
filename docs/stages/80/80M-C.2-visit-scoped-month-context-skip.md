# Stage 80M-C.2 — Visit-Scoped Month Context Skip

**Type:** Implementation  
**Verdict:** Built (local)  
**Date:** May 2026  
**Parent:** [80M-C.1 Month context skip audit](./80M-C.1-month-context-skip-behaviour-audit.md)

---

## Objective

Make **“Skip for now”** accurate: empty month context skip hides the prompt for the **current Reality visit only**, not the whole month.

**Rule:** Not now — not not this month.

---

## Scope

| File | Change |
|------|--------|
| `index.html` | Session skip helpers, dismiss/visibility/clear wiring |

---

## Session helpers

| Helper | Key | Purpose |
|--------|-----|---------|
| `geodeSetMonthContextSkipVisit` | `geode_month_context_skip_visit` | Set on empty skip |
| `geodeGetMonthContextSkipVisit` | — | Read skip flag |
| `geodeClearMonthContextSkipVisit` | — | Clear on leave Reality / save / month rollover |

**No localStorage schema change.** No new `S` fields.

---

## `geodeDismissMonthContextForNow`

| Context | Behaviour |
|---------|-----------|
| **Empty** (`!hasAny`) | Set visit skip only; no `dismissedYm`; no `save()`; no toast |
| **Has saved text** | Unchanged — `dismissedYm = currentYM()`, `save()`, toast |

---

## Visibility

`geodeRealityMonthContextSectionHtml` hides empty section when:

```js
!mc.hasAny && (mc.dismissedThisMonth || geodeGetMonthContextSkipVisit())
```

Legacy `dismissedYm` from prior builds still respected.

---

## Clear visit skip

- `geodeExitRealitySurface`
- `geodeNavigateToReflectionFromReality` (leave Reality sub-surface)
- `goTab` — leaving Money while on Reality/Reflection, or Money tab reset from Reality/Reflection
- `geodeSaveMonthContextFromReality`
- `geodeNormalizeMonthContext` — month rollover
- `rMoney` — when `msub` is not `reality` or `reflection`

---

## Unchanged

Reflection, Ask Beynd, Plan, Home, service worker, coaching.json, carry-forward (80E-D).

---

## Manual QA

1. Empty context → **Skip for now** → section hidden on Reality
2. Exit Reality → return → section visible again
3. Empty skip → no Reflection entry
4. Meaningful save → Reflection entry still works
5. Refresh after skip → section stays hidden until leave Reality (sessionStorage)
