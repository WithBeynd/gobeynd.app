## Changes

### Top-level `density`

```js
density: atmosphere === 'calm'
  ? 'light'
  : (atmosphere === 'room' || atmosphere === 'recovering' ? 'roomy' : 'standard')
```

### Block schema

Each block includes `source`:

| Block id | `source` |
|----------|----------|
| `what_matters_this_month` | `monthContext` |
| `month_so_far` | `atmosphere` |
| `background` | `heldBack` |
| `plan_protecting` | `planFocus` |

### `sanitizeCopy`

- Strips HTML tags from user-entered text before dash/whitespace normalisation.

### `maxBlocks`

Hard-capped at **3** (`Math.min(3, …)`).

### `audit` flags (all `true`)

- `shadowOnly`, `noUi`, `noRoutes`, `noStorage`, `noCalculations`

### `noGo`

- `autoOpen: false`
- `noCharts`, `noAmountsTable`, `noTrafficLights`, `noRecommendations`, `noRawInternals` — all `true`

## Unchanged

Block copy strategy, block ids, snapshot / visual model / entry gate helpers, Reality save, UI, routes, CSS, storage, SW, coaching.json.
