# Stage 80G-B.3 — Visual Model No-Go Contract Completion

**Type:** Metadata-only hardening  
**Verdict:** Built (local)  
**Date:** May 2026  
**Parent:** [80G-B.2 contract completion](./80G-B.2-visual-model-contract-completion.md)

---

## Objective

Add **`noGo`** metadata to `geodeStabilityRoomVisualModel` so future UI consumers inherit explicit product constraints without inferring them from audit comments.

**Not UI.** No atmosphere, section, entry copy, or snapshot changes.

---

## Change

Top-level field on visual model return object:

```js
noGo: {
  trafficLights: true,
  autoOpen: false,
  notificationDrawer: false
}
```

| Flag | Meaning |
|------|---------|
| `trafficLights` | **true** = forbid green/amber/red framing for Reflection Room |
| `autoOpen` | **false** = room must not open automatically |
| `notificationDrawer` | **false** = no notification-drawer entry pattern |
