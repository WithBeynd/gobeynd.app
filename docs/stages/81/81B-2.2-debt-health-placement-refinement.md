## Implementation

### 1. Remove duplicate **Review provider balance** button

Removed from `geodeDebtHealthCardHtml`. That button called `openDebtModal(id)` — identical to header **Edit**. Provider review path: **Edit → Save Debt**.

### 2. Compact muted inline copy

Replaced bordered “Debt health” box with 1–2 lines under the balance/APR grid:

- Line 1: `{APR type label} · {provider review status}`
- Line 2 (stale or never reviewed only): existing neutral nudge from `geodeDebtProviderReviewNudgeCopy`

Styling: `font-size:9px`, `color:var(--tx-3)` — no title, border, background, badges, or warning colours.

### 3. Conditional visibility — `geodeDebtShouldShowHealthOnCard(debt, state)`

Show inline copy when **any** of:

| Trigger | Source |
|---------|--------|
| Never reviewed | `lastProviderReviewAt` empty |
| Stale review | `geodeDebtDaysSinceProviderReview` > threshold (45d unknown/variable, 60d fixed) |
| Unknown or variable APR | `interestType` |
| Paid payment since last review | `geodeDebtPaidPaymentSinceProviderReview` (existing payments + review dates) |

**Hide** when: fixed APR, fresh provider save, no paid debt payment since that save.

### Helpers added

- **`geodeDebtPaidPaymentSinceProviderReview(debt, state)`**
- **`geodeDebtShouldShowHealthOnCard(debt, state)`**

## Validation

```js
typeof geodeDebtShouldShowHealthOnCard === 'function';
typeof geodeDebtPaidPaymentSinceProviderReview === 'function';

// New debt — shows inline (no review)
geodeDebtShouldShowHealthOnCard({ id: 'x', interestType: 'unknown' }, S);

// Fixed + saved today — hidden unless paid payment since save
geodeDebtShouldShowHealthOnCard(
  { id: 'x', interestType: 'fixed', lastProviderReviewAt: geodeTodayLocalISO() },
  S
);

geodeDebtHealthCardHtml({ id: 'x', interestType: 'unknown', balance: 1000 }, S).indexOf('Review provider balance') < 0;
geodeDebtHealthCardHtml({ id: 'x', interestType: 'unknown', balance: 1000 }, S).indexOf('Debt health') < 0;
```

Manual QA:

1. New debt → one muted line under metrics (no box, no extra button)
2. Edit → save with fixed APR → line hides if no paid payment since save
3. Mark debt payment paid → line reappears for that debt
4. Edit still opens full provider-review modal copy
5. **Pay toward this debt** unchanged
